(Internship) NHS 360° VR Project

Language: C# (Unity)

Description:

                     The idea was to create a VR and mobile app in which doctors could post a 360° video to a website and patients could watch and provide feedback.
Throughout the process, I had to learn the website pipeline and how to send and access data that would later be utilised as variables to change questions and activate / disable features. All data collected from app usage is later transferred to a cloud document, allowing the doctor to access and compare progress while also evaluating the rating of the videos uploaded.

Implementation:
          • Login page
          • Patient feedback page         
          • Data tracking
          • Video rating page
          • Options that can change the app's structure from the website.
